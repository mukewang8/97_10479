<p><a title="查看与 Spring 相关的文章" target="_blank">Spring</a>框架的核心说白了，就2点，就是Ioc和AOP。Ioc就是统一的Java的Bean类的管理工具，“控制反转”，也就是著名的“好莱坞原则”。AOP的话，就是面向切面编程，使你专注于自己业务代码的实现，至于时间统计，缓存添加等于核心业务无关代码，可以通过注解的方式，以切面编程的方式实现。</p>
